# 🤖 Miniguia: O Futuro dos Agentes de IA em 2026

## 🎯 Contexto e Objetivos
Este repositório contém o resultado do meu estudo sobre **Agentes de IA**, utilizando o **NotebookLM** como ferramenta de aprendizagem ativa. O objetivo principal foi entender como a IA evoluiu de modelos de chat para agentes autônomos que podem executar tarefas, especialmente no contexto de análise de dados (SQL e Python).

## 📚 Curadoria de Fontes
Utilizei fontes atualizadas (2025-2026) para garantir a relevância técnica:
1. **Relatório Distrito (2025):** Panorama do mercado brasileiro de Agentes de IA.
2. **Whitepaper ABFP (2025):** Agentes de IA como força de trabalho e autonomia.
3. **Guia Técnico de Deploy (2025):** Ciclo de vida de agentes do zero ao deploy.
4. **IBM Think (2026):** Tendências globais e arquitetura de sistemas agenticos.
5. **DataEX (2026):** Projeções de adoção de IA nas grandes empresas.

## 🧠 Engenharia de Prompts 
Durante o uso do NotebookLM, explorei diferentes abordagens:
- **Prompt Estratégico:** *"Com base nas fontes, qual a diferença crítica entre um Agente de IA e um LLM tradicional no que diz respeito ao uso de ferramentas (Tool Use)?"*
- **Dificuldade (Troubleshooting):** Inicialmente, os resumos eram genéricos. Ajustei os prompts para focar em casos de uso práticos para Analistas de Dados, o que trouxe respostas muito mais ricas sobre automação de queries SQL.

## 📖 Miniguia de Estudo (Entrega Final)
### Resumo Estruturado
- **O que são Agentes:** Sistemas capazes de perceber o ambiente, raciocinar e agir usando ferramentas externas.
- **Diferencial de 2026:** A capacidade de planejar etapas complexas antes de executar a primeira ação (Reasoning Loops).

### Glossário
- **Reasoning (Raciocínio):** Processo de deliberação da IA antes da resposta.
- **Tool Use:** Capacidade de chamar APIs, rodar códigos Python ou consultar bancos SQL.

- *"Resuma este documento focando apenas nas implicações para a área de Engenharia de Dados."*
- *"Crie um roteiro de estudos de 5 dias sobre este tema para um profissional de transição de carreira."*
